# MyAnswerSkinCare

A frontend e-commerce web application built with **React** and **TypeScript**.

This project focuses on building a skincare shopping experience with modular component design, page-based structure, and reusable frontend logic.

---

##  Tech Stack

- React
- TypeScript
- SCSS
- React Router
- JavaScript
- HTML5
- CSS3
- Node.js
- npm

---

##  Features

- Product listing page
- Product detail page
- Shopping cart flow
- Checkout flow
- Wishlist feature
- Admin dashboard pages
- Reusable modal and form components
